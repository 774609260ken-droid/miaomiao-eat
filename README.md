# 淼淼今天吃什么了 ♡

两个人专属的可爱点单网页。

## 已实现

- 9 个品牌切换：茶百道、霸王茶姬、一点点、蜜雪冰城、瑞幸咖啡、喜茶、乐乐茶、奈雪的茶、星巴克
- 45 款代表性产品
- 甜度、冰量和留言
- 她的点单端
- 男朋友工作台
- 接单、已买好、已送达三段状态流转
- 本地订单历史和累计参考金额
- 手机端响应式布局

## 当前阶段

当前订单保存在浏览器 localStorage 中，适合单机演示。两台手机实时同步需要下一步接入云数据库和身份识别。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

项目已包含 GitHub Pages 部署工作流。仓库开启 Pages 后，推送到 `main` 分支即可自动构建并发布。
